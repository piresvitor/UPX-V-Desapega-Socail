# 🚀 Desapega Social - Conectando Generosidade e Necessidade

O **Desapega Social** é uma solução de Transformação Digital focada em economia circular e impacto social na cidade de Sorocaba/SP. Nosso objetivo é facilitar a doação de itens usados (móveis, roupas e utensílios), garantindo que cheguem primeiro a quem mais precisa.

---

## 💡 A Proposta
Diferente de marketplaces comuns, o Desapega Social utiliza um **sistema de prioridade inteligente**:
* **Exclusividade Social:** Beneficiários de baixa renda cadastrados possuem 24h de exclusividade para solicitar novos itens.
* **Acesso Aberto:** Após o período de carência, os itens ficam disponíveis para o público geral, evitando o desperdício (ODS 12).
* **Logística Integrada:** Conexão direta com freteiros locais para viabilizar a entrega de itens de grande porte.

## 🌍 ODS Atendidas (ONU)
* **ODS 1:** Erradicação da Pobreza
* **ODS 10:** Redução das Desigualdades
* **ODS 12:** Consumo e Produção Responsáveis

## 🛠️ Stack Tecnológica (Prevista)
* **Backend:** Node.js (Fastify/Express)
* **Banco de Dados:** PostgreSQL (PostGIS para geolocalização)
* **Mobile:** React Native
* **Arquitetura:** REST API com autenticação JWT e validação de documentos via OCR.

## 👥 Público-Alvo
1. **Doadores:** Pessoas e empresas com itens em bom estado.
2. **Beneficiários:** Público geral e cidadãos em vulnerabilidade (com prioridade).
3. **Freteiros:** Profissionais autônomos para logística social.
4. **ONGs:** Parceiros institucionais para validação e pontos de coleta.
